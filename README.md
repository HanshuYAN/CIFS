# CIFS

CIFS: Improving Adversarial Robustness of CNNs via Channel-wise Importance-based Feature Selection  
Hanshu Yan, Jingfeng Zhang, Gang Niu, Jiashi Feng, Vincent Y. F. Tan, Masashi Sugiyama


## Requirements
Python = 3.6  
Pytorch = 1.6  
CUDA = 10.1  
Advertorch = 0.2.3
